# siem casero

## Documentacion de el trabajo final que he realizado.



## 📌 Descripción del Proyecto

Diseño e implementación de un sistema **SIEM (Security Information and Event Management)** de bajo coste utilizando hardware Raspberry Pi y un Intel NUC. El sistema permite la detección de amenazas en redes locales mediante un sensor IDS (Suricata) que envía alertas a un stack ELK para su visualización y análisis.
